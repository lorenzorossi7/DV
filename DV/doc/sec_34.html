Saving/Loading registers to/from disk

Registers can be saved to disk by first selecting them in the DV main window, then choosing the Write selected SDF(s) to file under the File menu. For each register that you have selected, a dialogue window will pop up, in turn, where you can enter the disk location and file name where you want to save the register. The register will be saved in SDF format. Single registers can also be loaded into the DV using the Read SDF from file option under the File menu (or using sdftodv).
